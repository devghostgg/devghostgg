## 👨‍💻 Sobre Mim

Sou um desenvolvedor **Full Stack** apaixonado por criar sistemas de alto desempenho. Tenho uma vasta experiência no ecossistema de **Grand Theft Auto V (FiveM)** e **Red Dead Redemption 2 (RedM)**, desenvolvendo desde frameworks do zero até sistemas administrativos, interfaces (NUI) interativas (Vue.js/React/HTML) e integrações avançadas com banco de dados.

Também atuo fortemente na criação de **Bots para Discord** super otimizados, desenvolvimento de **Sistemas Web (SaaS)**, **Landing Pages** de alta conversão e **Sistemas IPTV**.

🛒 Sou o **Dono e Desenvolvedor** da **[South Network](https://github.com/southnetworkgg)**, minha loja oficial onde crio e disponibilizo sistemas inovadores e profissionais para a comunidade de FiveM, RedM e Discord.

### 🎯 Especialidades

- 🎮 **FiveM & RedM:** Scripts avançados, UI responsiva (NUI), máxima otimização (0.00ms resmon), e frameworks customizados.
- 🤖 **Bots para Discord:** Automação, sistemas de ticket, logs avançados, integração com banco de dados e bots de vendas/whitelist automáticos.
- 🌐 **Sistemas Web:** Dashboards (Painéis Administrativos), Landing Pages, e plataformas SaaS completas.
- 📺 **IPTV:** Sistemas de painéis, gerenciamento de assinaturas e streaming.
